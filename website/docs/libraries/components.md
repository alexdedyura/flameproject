# Компоненты open.mp

Компоненты (`components/*.dll`) — это нативные подсистемы движка open.mp, поставляемые вместе с сервером.
Их **не редактируют** — гейммод просто опирается на их нативы.

## Основные компоненты

Движок предоставляет нативные подсистемы, среди которых:

- **Actors** — актёры/NPC-персонажи;
- **Vehicles** — транспорт;
- **Objects** — объекты;
- **TextDraws** — текстовые рисунки (UI);
- **Pickups** — пикапы;
- **Dialogs** — диалоги;
- **GangZones** — зоны;
- **Menus** — меню;
- **NPCs** — неигровые персонажи;
- **Recordings** — записи (для NPC-маршрутов);
- **Checkpoints** — чекпоинты;
- **Classes** — классы игроков (выбор персонажа).

## Отличие от плагинов

| | Компоненты open.mp | Legacy-плагины |
| --- | --- | --- |
| Каталог | `components/*.dll` | `plugins/*.dll` |
| Что это | Нативные подсистемы движка | Внешние Pawn-плагины (SA-MP-наследие) |
| Подключение | Идут с сервером | Через `pawn.legacy_plugins` в `config.json` |
| Редактирование | Нет | Нет (read-only) |

Список legacy-плагинов — на странице [Плагины](plugins).

:::note
В сборку сервера (`.zip` из CI) попадают только `.dll` компонентов — без debug-символов `.pdb`.
:::
