# Транспорт

Крупнейшая подсистема гейммода. Агрегатор — `sources/vehicles/vehicles.inc`.

## Модули

| Файл | Назначение |
| --- | --- |
| `vehicles_core.pwn` | Ядро транспорта, enum'ы, спавн. |
| `vehicles_player.pwn` | Владение транспортом игроком. |
| `vehicles_shop.pwn` | Автосалон / покупка (`VShop_*`). |
| `vehicles_keys.pwn` | Ключи (`VKeys_*`). |
| `vehicles_sharing.pwn` | Каршеринг (`VSharing_*`). |
| `vehicles_engine.pwn` | Двигатель и топливная система. |
| `vehicles_tax.pwn` | Налог на транспорт. |
| `vehicles_numbers.pwn` | Номерные знаки. |
| `vehicles_params.pwn` | Параметры транспорта. |
| `vehicles_comp.pwn` | Компоненты / покраска. |
| `vehicles_garage.pwn` | Гараж. |
| `vehicles_stats.pwn` | Статистика транспорта. |
| `speedometer.pwn` | Спидометр (`Speed_*`, `CreateSpeedometerPTD`). |

## Основные механики

- **Владение и покупка** — покупка машин в автосалоне (`VShop`), закрепление за игроком.
- **Ключи** (`VKeys`) — доступ к запуску/открытию транспорта.
- **Каршеринг** (`VSharing`) — совместное использование транспорта.
- **Двигатель и топливо** — ручное управление двигателем/фарами (`use_manual_engine_and_lights`
  в `config.json`), расход топлива (`SetFuelSystemCars` при инициализации).
- **Налог** — периодический налог на владение.
- **Номера, покраска, компоненты** — кастомизация.
- **Гараж** — хранение транспорта.
- **Спидометр** — TextDraw-интерфейс скорости (см. [Интерфейс](interface)).

## Инициализация

В `OnGameModeInit()` последовательно поднимаются: автосалон (`VShop`), каршеринг (`VSharing`),
ключи (`VKeys`) и топливная система (`SetFuelSystemCars`).

## Callback'и

`flame.pwn` вручную диспетчеризует связанные с транспортом callback'и в per-module функции:

- `Vehicles_OnPlayerStateChange(...)` — реакция транспорта на смену состояния игрока;
- `Speed_OnPlayerStateChange(...)` — обновление спидометра.

## Хранение

Транспорт (владение, параметры) сохраняется в таблицу `vehicles` (см. [База данных](../database)).

## Связанные системы

- Вызов транспорта доступен со [Смартфона](phone).
- Интерьеры автосалона и DMV — в [Объектах](objects).
