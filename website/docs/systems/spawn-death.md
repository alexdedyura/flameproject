# Спавн и смерть

Система возрождения игрока и обработки смерти.

## Модули

| Файл | Назначение |
| --- | --- |
| `player/spawn.inc` | Спавн — появление/возрождение игрока в мире. |
| `player/death.inc` | Обработка смерти персонажа. |

## Как это работает

- После авторизации и выбора персонажа игрок **спавнится** в мире (`spawn.inc`).
- При гибели срабатывает логика **смерти** (`death.inc`) — учёт статистики (deaths),
  обработка возрождения.

## Связанные системы

- Вход в игру начинается с [Авторизации](authorization).
- Статистика kills/deaths хранится в аккаунте (таблица `accounts`, см. [База данных](../database)).
- Настройка `death_drop_amount` в `config.json` управляет выбросом денег при смерти
  (по умолчанию `0`).
