# Квесты и RP-рейтинг

Прогрессия персонажа: задания и рейтинг отыгрыша роли.

## Модули

| Файл | Назначение |
| --- | --- |
| `player/quests.inc` | Квесты — задания для игрока. |
| `player/rp_rating.inc` | RP-уровень и опыт (rating). |

## Как это работает

- **Квесты** дают игроку цели и продвигают прогрессию персонажа.
- **RP-рейтинг** — уровень и опыт, отражающие вовлечённость игрока в ролевой процесс.

## Хранение

RP-опыт и уровень входят в аккаунт игрока (таблица `accounts`, поля RP-опыта/уровня) —
см. [База данных](../database).

## Связанные системы

- RP-чат (`/me`, `/do` и др.) — основной инструмент отыгрыша, см. [Команды](commands).
- Работы (`job_horizon`) дают ещё один способ прогрессии, см. [Мир и системы](world).
