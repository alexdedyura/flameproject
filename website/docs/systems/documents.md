# Документы и лицензии

Система персональных документов игрока.

## Модули

| Файл | Назначение |
| --- | --- |
| `player/documents.inc` | Общая логика документов игрока. |
| `player/id_card.inc` | ID-карта (удостоверение личности). |
| `player/driving_licenses.inc` | Водительские права. |

## Хранение

Документы игрока сохраняются в таблицу `documents` (см. [База данных](../database)).

## Связанные системы

- **Водительские права** логически связаны с [Транспортом](vehicles) — для управления транспортом
  и с DMV (интерьер `objects/dmv.inc`, где права можно получить).
- **ID-карта** используется как удостоверение личности в RP-взаимодействиях.
