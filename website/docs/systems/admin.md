# Админка

Административные инструменты. Агрегатор — `sources/player/admin/admin.inc`.

## Что входит

- **Админ-инструменты** — команды и функции модерации/управления сервером.
- **Spectate** — наблюдение за игроками.

## Уровни доступа

Admin-уровень игрока хранится в аккаунте (таблица `accounts`, см. [База данных](../database)) —
он определяет доступность админ-команд.

## Связанные механики

- Часть админ-команд находится в `commands/main.inc` (см. [Команды](commands)).
- Управление сервером на низком уровне — через **RCON** (пароль в `config.json`,
  см. [Конфигурация](../getting-started/configuration)).

:::note
`allow_teleport` для RCON по умолчанию выключен (`config.json` → `rcon.allow_teleport: false`).
:::
