# Требования

Для запуска и разработки Flame Project понадобится следующее.

## Для запуска сервера

| Компонент | Назначение |
| --- | --- |
| **Windows** | Основная платформа. Сервер запускается через `omp-server.exe`. Linux-сборка запланирована, но пока не поддерживается. |
| **MySQL** | Хранение аккаунтов, инвентаря, транспорта, банка и др. База называется `flame`. |
| **libmysql.dll** | Клиентская библиотека MySQL — уже поставляется в репозитории рядом с сервером. |

Готовые бинарники движка (`omp-server.exe`), нативные компоненты open.mp (`components/*.dll`) и
legacy-плагины (`plugins/*.dll`) уже включены в репозиторий — отдельно скачивать их не нужно.

## Для сборки гейммода

| Компонент | Назначение |
| --- | --- |
| **qawno / pawncc** | Компилятор Pawn open.mp. Лежит в `qawno/` и вызывается из `compile.bat`. |
| **Инклюды** | `open.mp`, `YSI`, `izcmd`, `sscanf2`, `streamer`, `a_mysql`, `rustext` — все в `qawno/include`. |

:::info
Отдельного шага линтинга или юнит-тестов в проекте нет. «Сборка прошла» означает, что компилятор Pawn
создал `gamemodes/flame.amx` без новых ошибок и предупреждений.
:::
