# Flame Project

**Flame Project** — это Open Source RPG-гейммод для [open.mp](https://www.open.mp/) (open multiplayer),
написанный на **Pawn**. Изначально проект разрабатывался для SA:MP, а сейчас полностью переведён на
open.mp через слой совместимости `SAMP_COMPAT`.

Весь гейммод компилируется в единственный `flame.amx` из одной точки входа — `sources/flame.pwn`.

:::tip С чего начать
- Хотите **поднять сервер** — переходите к разделу [Установка](getting-started/requirements).
- Хотите **понять устройство кода** — начните с [Архитектуры](architecture/overview).
- Ищете конкретную механику — смотрите [Игровые системы](systems/authorization).
:::

## Что внутри

Игровой процесс построен вокруг классической RPG-механики (San Fierro как основная карта):

| Направление | Кратко |
| --- | --- |
| **Игрок** | Авторизация, аккаунт и сохранения, инвентарь, банк, документы и лицензии, квесты, RP-рейтинг, спавн и смерть. |
| **Смартфон (iPhone)** | Интерфейс телефона, SMS с историей, обои, часы, вызов транспорта. |
| **Транспорт** | Владение и покупка, ключи, каршеринг, топливо, налог, номера, покраска, гараж, спидометр. |
| **Мир и системы** | Автобусные остановки, пикапы входов/выходов, погода, работы. |
| **Объекты** | Большой набор статичных интерьеров и локаций (банк, госпиталь, PD, DMV, аэропорт и др.). |
| **Интерфейс** | TextDraw-UI: спидометр, инвентарь, iPhone, загрузочный экран и др. |
| **Команды** | ~287 чат-команд, сгруппированных по темам. |

## Технологический стек

| Слой | Что используется |
| --- | --- |
| Платформа | open.mp (open multiplayer), запуск через `omp-server.exe` |
| Язык | Pawn (`SAMP_COMPAT` + `#include <open.mp>`) |
| Библиотеки / инклюды | YSI (`y_hooks`, `y_iterate`), `izcmd`, `sscanf2`, `streamer`, `a_mysql`, `rustext` |
| Плагины | `crashdetect`, `jit`, `streamer` (Incognito), `sscanf`, `mysql`, `pawn-memory`, `rustext` |
| Компоненты | нативные подсистемы open.mp: Actors, Vehicles, Objects, TextDraws, Pickups, Dialogs и др. |
| База данных | MySQL, база `flame` |
